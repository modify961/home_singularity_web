# Home Singularity Web

家庭智能系统前端

## 环境要求

- Node.js 18.x LTS
- npm 9+

## 项目启动

```bash
# 安装依赖
npm install

# 开发模式
npm start

# 生产构建
npm run build
```

更多详细步骤：见 `docs/Getting-Started.zh-CN.md`。

## UI 组件
- [Material UI](https://mui.com/)（@mui/material）
- [Material Icons](https://mui.com/material-ui/material-icons/)（@mui/icons-material）
